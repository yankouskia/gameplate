## Bundle and runtime info

|                           |                                                                  |
| :------------------------ | :--------------------------------------------------------------- |
| **Bundle size (gzipped)** | ~3 KB (target: &lt; 4 KB hard cap in CI)                         |
| **Tree-shaking**          | `sideEffects: false` — only imported symbols land in your bundle |
| **Runtime dependencies**  | **None.**                                                        |
| **Modules**               | Dual ESM + CJS; `exports` map drives resolution                  |

Changelogs are auto-generated by [changesets](https://github.com/changesets/changesets) and
published as [GitHub Releases](https://github.com/yankouskia/gameplate/releases).
